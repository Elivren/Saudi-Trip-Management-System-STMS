 Saudi-Trip-Management-System-STMS # 🇸🇦 Saudi Trip Management System (STMS)

A full-stack web platform for managing tourism trips across Saudi Arabia.

 👥 User Roles
- **Tourist** – Browse and book trips & packages
- **Guide** – Manage assigned trips and bookings  
- **Admin** – Full control over users, trips, and reports

 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL (10+ tables, normalized schema)

 ✨ Features
- User authentication & role-based access
- Trip & package management
- Booking system with status tracking
- Admin dashboard with full oversight
- 8+ responsive UI screens

 🎓 Project Info

Faculty of Computing and Information Technology
